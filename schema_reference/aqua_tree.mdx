An Aqua tree is the root object of Aqua Protocol schema defination .
It wraps two object revsion and file index.

 * To understand revisions [read](/schema_reference/revision)
 * To understand File index [read](/schema_reference/file_index)


<Check>The term tree is used and not chain as the revisions can contain more than one child while chain assumes a one vs One relationship hence the use of the word tree</Check>
